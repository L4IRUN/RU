雲端同步設定

同步設定教學

1. 取得 GitHub Token:

  • 前往 GitHub Tokens (Classic)。
  • 隨意填寫 Note，並在下方權限清單中只勾選 gist。
  • 點擊最下方按鈕產生，並複製 ghp_ 開頭的密碼。

2. 取得 Gist ID:

  • 前往 GitHub Gist。
  • 隨意輸入檔名與內容，點擊 Create secret gist 建立。
  • 查看網址列，複製最後一串亂碼。
    (例如: gist.github.com/帳號/[這串就是ID])

尚未設定雲端同步

請先設定 GitHub Token 與 Gist ID 才能使用雲端備份功能。
